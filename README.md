# Top Up Business Website

A professional landing page for Top Up - showcasing our all-in-one platform for communities, businesses, digital wallet, and location discovery.

## Features

- Modern, responsive design
- Smooth scrolling navigation
- Feature showcase
- Contact information
- Easy to deploy

## Deployment Options

### Option 1: Netlify (Recommended - Easiest)

1. Go to [netlify.com](https://www.netlify.com)
2. Sign up/login (free)
3. Drag and drop the `dayly_site` folder onto Netlify
4. Get your live URL instantly (e.g., `https://topup-site.netlify.app`)

### Option 2: Vercel

1. Go to [vercel.com](https://www.vercel.com)
2. Sign up/login (free)
3. Click "New Project"
4. Import your folder or connect GitHub
5. Deploy and get your URL

### Option 3: GitHub Pages

1. Create a GitHub repository
2. Upload these files
3. Go to Settings > Pages
4. Select main branch
5. Your site will be at `https://yourusername.github.io/dayly_site`

### Option 4: Firebase Hosting

```bash
# Install Firebase CLI
npm install -g firebase-tools

# Login
firebase login

# Initialize
firebase init hosting

# Deploy
firebase deploy
```

## Customization

- Edit `index.html` to change content
- Edit `styles.css` to change colors/styling
- Edit `script.js` for interactive features

## For PayFast Verification

Once deployed, you'll get a public URL like:
- `https://topup-site.netlify.app`
- `https://topup.vercel.app`
- Or your custom domain

Send this URL to PayFast for verification.

## Quick Start

1. Open `index.html` in a browser to preview locally
2. Deploy using one of the options above
3. Share the URL with PayFast
